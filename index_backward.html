An Interactive Visualization of the Housing Characteristic Dependencies in ResStock

ResStock is built upon the conditional probabilites of the housing characteristics. Each housing characteristic has a set of dependencies and dependants. In this dependency wheel, each chord in the disc represents a dependency. The thin part of the cord represents a dependency to the thicker part of the cord. Try hovering on packages to mask other dependencies. All rendering is done client-side, in JavaScript. Built with d3.js

Dependency Wheel

Creation Details

The data for the dependency wheel was produced by a python script. The script goes through the Housing Characteristics directory in a ResStock project folder and scrapes the dependencies from each housing characteristic. The data is interpreted into an adjacency matrix, which in turn is used to produce a json dictionary. The json dictionary is then used to write the composer.json and the composer.lock files necessary for the dependency wheel.

Reference
